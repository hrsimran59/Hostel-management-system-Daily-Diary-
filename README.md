# Hostel Management System

This project is developed as part of six months industrial training / project work.

## Project Overview
The Hostel Management System is a web-based application designed to automate and manage hostel operations such as student registration, room allocation, fee management, and complaint handling.

## Technologies Used
- HTML, CSS, JavaScript, Bootstrap
- Node.js, Express.js
- MongoDB
- Firebase Authentication

## Features
- Student Registration and Login
- Room Allocation and Availability Tracking
- Online Complaint Submission and Tracking
- Fee Payment and Status Management
- Admin Dashboard for Hostel Management
